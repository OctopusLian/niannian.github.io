无穷等比数列求和的方法

公比$q = a_n / a_n-1$

求和 $S_n = a_1 + a_1q^2 + a_1q^3 + … + a_n*q^n-1$ (1)式

两边同时乘以q，qS_n = a_1q^2 + a_1q^3 + … + a_nq^n-1 + a_n*q^n$ (2)式

令(2)式 - (1)式： qS_n - Sn = a_1*q^n - a_1
S_n(q-1) = a_1(q^n - 1)
S_n = a_1(q^n - 1) / q - 1

当 n -> 无穷大 ，q^n -> 0， S_n ≈ a_1*(0-1)/q - 1
S_n ≈ a_1 / 1- q

布隆过滤器-Bloom filter

Bloom filter vs Hash Table

一个很长的二进制向量和一系列随机映射函数。布隆过滤器可以用于检索一个元素是否在一个集合中。

优点：空间效率和查询时间都远远超过一般的算法。

缺点：有一定的误识别率和删除困难。

结论：如果这个元素在布隆过滤器查不到，那肯定不存在；如果查得到，那可能存在。

案例

1，比特币网络
2，分布式系统（Map-Reduce）——Hadoop、Search Engine
3，Redis缓存
4，垃圾邮件、评论等的过滤
